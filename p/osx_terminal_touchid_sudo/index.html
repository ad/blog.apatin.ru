sudo через TouchID/Apple Watch в OS X

Чтобы вместо ввода пароля для sudo прикладывать палец к TouchID нужно отредактировать файл

sudo nano /etc/pam.d/sudo

добавив второй строкой:

auth sufficient pam_tid.so
